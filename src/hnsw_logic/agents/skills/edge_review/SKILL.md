# Edge Review

Review candidate edges with a second-pass mindset.

- Prefer durable, reusable retrieval edges over broad topical relatedness.
- Down-rank or reject pairs when the evidence is weak, generic, or mostly about co-usage.
- Pay attention to uncertainty, directionality, and risk flags before approving an edge.
- If the pair is related but not retrieval-useful, set `canonical_relation` to `none`.
