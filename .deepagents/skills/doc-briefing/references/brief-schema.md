# Brief Schema

Required fields:

- `doc_id`
- `title`
- `summary`
- `entities`
- `keywords`
- `claims`
- `relation_hints`
- `metadata`

Profiling guidance:

- Prefer neutral analytical summaries over raw quotations.
- Preserve stance, topic family, topic cluster, and retrievable bridge terms.
- Use excerpt-first evidence for high-risk or heavily argumentative passages.
- Only retain exact phrasing when it materially changes the indexing decision.
