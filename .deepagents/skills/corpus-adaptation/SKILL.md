# Corpus Adaptation

Use this skill to classify corpus behavior and choose a safe indexing posture.

## Workflow

1. Infer corpus style from titles, claims, lexical agreement, and graph potential.
2. Prefer utility-first conservative indexing for low-structure corpora.
3. Allow stronger bridge exploration for scientific and technical corpora.
4. Record failure patterns for future batches.
